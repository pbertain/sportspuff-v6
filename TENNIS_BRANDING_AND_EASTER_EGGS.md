# Tennis Branding and Easter Eggs

## Goal

Create a polished tennis-themed visual system for SportsPuff that feels professional at first glance, but rewards exploration with playful easter eggs.

Use two visual layers:

1. Official-inspired tournament assets for normal UI use.
2. Custom SportsPuff / Puhfph mascot assets for hidden rewards, 404 pages, achievements, theme transitions, and seasonal fun.

## Core Tournament Set

- ATP
- WTA
- Wimbledon
- Roland Garros / French Open
- US Open
- Australian Open

## Official-Inspired Tennis Balls

Each tournament should have a professional but cartoonish tennis ball with a clean white or transparent background.

Suggested colors:

| Asset | Ball Color |
|---|---|
| ATP | deep navy |
| WTA | purple |
| Wimbledon | grass green |
| Roland Garros / French Open | clay orange |
| US Open | deep US Open navy |
| Australian Open | bright AO blue |

Use the actual uploaded/source logo references only as inspiration unless project licensing allows official marks.

## Easter Eggs

### Career Grand Slam Mode

If users click the tournament balls in the Grand Slam order:

1. Australian Open
2. Roland Garros
3. Wimbledon
4. US Open

Unlock:

> Career Grand Slam Mode

Theme changes to gold accents.

### Tennis Completionist

If users collect ATP, WTA, and all four Grand Slam stamps:

> Tennis Completionist

### Puhfph Open

After unlocking all achievements, reveal a hidden fifth tournament:

- Name: Puhfph Open
- Location: Las Lunas, NM
- Surface: Colorblind Hardcourt
- Ball: rainbow tennis ball with puff-cloud mark
- Prize: bragging rights

### Rain Delay Mode

On Wimbledon pages, random rare effect:

> Play suspended due to rain.

Light rain animation appears briefly.

### Clay Dust Cursor

On Roland Garros pages, dragging the mouse leaves subtle clay dust trails.

### GOAT Detector

Hidden goat icon in footer.

Click it:

> Initiating GOAT debate...

Show collectible GOAT cards, but never declare a winner.

### Tennis Ball Physics

Tournament balls can squish, rotate, and eventually fly across the screen if clicked repeatedly.

## Theme Switcher

Tournament balls should act as theme buttons:

- Wimbledon: grass theme
- Roland Garros: clay theme
- US Open: hard-court navy theme
- Australian Open: AO blue theme

## 404 Page Concepts

Tournament-specific 404 pages:

- Wimbledon: lost groundskeeper
- Roland Garros: confused rooster
- US Open: lost NYC pigeon
- Australian Open: bewildered kangaroo

Message examples:

- Player not found.
- Match point missed.
- This draw has no such player.
- The ball was in, but this page is out.

## Accessibility

- Maintain colorblind-safe contrast.
- Do not rely only on color to communicate tournament identity.
- Add alt text for all mascot and tournament ball assets.
- Theme switching must preserve readable text contrast.
